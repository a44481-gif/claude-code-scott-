# ⏳ 任務 B：設置 YouTube OAuth（5分鐘，未來永久全自動）

**在另一個瀏覽器標籤頁，同時打開：**
```
https://console.cloud.google.com
```

### 步驟 1：新建項目
- 點右上角「選擇項目」
- 點「新建項目」
- 名稱填：**anime-ops**
- 位置：**無組織**
- 點「創建」

### 步驟 2：啟用 API
- 左側：API和服務 → 啟用 API 和服務
- 搜：「YouTube Data API v3」
- 點擊結果 → 啟用

### 步驟 3：創建 OAuth 憑證
- 左側：憑證 → +創建憑證 → OAuth 用戶端 ID
- 應用程式類型：桌面應用
- 名稱：anime-ops
- 點「創建」

### 步驟 4：下載文件
- 看到「您的 OAuth 用戶端」彈窗
- 點「下載 JSON」

### 步驟 5：放到正確位置
- 把下載的文件**改名**為：`client_secrets.json`
- 把文件移動到：
  ```
  D:\claude mini max 2.7\anime_overseas\client_secrets.json
  ```
