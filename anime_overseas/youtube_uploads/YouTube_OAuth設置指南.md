# YouTube OAuth 設置 · 一步完成

## 只需要做一次，未來永久全自動

---

### 第一步：下載並運行我的自動化設置工具

在你的電腦上（任何設備都可以），用瀏覽器打開：

```
https://console.cloud.google.com
```

跟著下面的截圖說明操作（5分鐘）：

---

### 步驟 1：創建項目（1分鐘）

1. 點右上角 **「選擇項目」**
2. 點 **「新建項目」**
3. 名稱填：**anime-ops**
4. 位置選： **無組織**
5. 點 **「創建」**

---

### 步驟 2：啟用 API（1分鐘）

1. 左側選單 → **「API和服務」**
2. 點 **「啟用 API 和服務」**
3. 搜尋框輸入：**YouTube Data API v3**
4. 點擊結果 → 點 **「啟用」**

---

### 步驟 3：創建 OAuth 憑證（2分鐘）

1. 左側選單 → **「憑證」**
2. 點 **「+ 創建憑證」**
3. 選 **「OAuth 用戶端 ID」**
4. 應用程式類型選：**「桌面應用」**
5. 名稱填：**anime-ops**
6. 點 **「創建」**

---

### 步驟 4：下載並放置文件（30秒）

1. 看到彈窗 **「您的 OAuth 用戶端」**
2. 點 **「下載 JSON」** 按鈕
3. 把下載的文件**改名**為：`client_secrets.json`
4. 把這個文件發到你的電腦，保存到：

```
D:\claude mini max 2.7\anime_overseas\client_secrets.json
```

---

### 步驟 5：告訴我完成！

完成後，回到這裡告訴我。

**我收到後立刻：**
- ✅ 測試 OAuth 連接
- ✅ 自動上傳全部 5 個剪輯到你的 YouTube
- ✅ 為你訂閱者顯示為你本人上傳的視頻
- ✅ 以後每個新視頻都全自動上傳
