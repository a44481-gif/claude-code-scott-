## 1. 账号搭建 SOP

### Day 1-3：账号注册

**YouTube 账号 × 3**
```
步骤：
1. 浏览器无痕模式注册 Google 账号（每个账号独立IP/设备）
2. 创建 YouTube 频道
   - 频道名格式：{动漫名}+{语言} (例: DouluoDalu EN)
   - 头像：动漫角色高清图（版权注意）
   - Banner：统一品牌视觉
3. 完善频道信息：
   - 关于页面填写频道定位
   - 添加社交链接
   - 设置邮箱（用于接收通知）
4. 开通 YouTube Studio
```

**TikTok 账号 × 3**
```
步骤：
1. 手机号/邮箱注册（与YouTube账号分开）
2. 完善主页：
   - 用户名：动漫相关+语言
   - 头像：动漫风格统一
   - Bio：英文简介+订阅引导
3. 账号定位标签（账号名前20个粉丝会定义你的账号方向）
```

**Fiverr / Upwork 账号**
```
步骤：
1. 注册账号，设置为自由职业者
2. 创建服务 Gig：
   - 动漫解说配音（$50-200/条）
   - 字幕翻译（$30-80/条）
   - 视频剪辑（$100-300/条）
3. 完善 Portfolio（放3-5个样例作品）
4. 首单定价：市场价70%（吸引初始评价）
```

## 2. 每日运营 SOP

### 每日时间表

```
06:00-07:00  查看昨日数据（播放量、收益、新增粉丝）
07:00-08:00  发布当日内容（3-5条）
08:00-09:00  互动：回复评论、私信、与其他账号互动
09:00-11:00  内容制作：剪辑+字幕+封面
11:00-12:00  版权筛查 + 合规检查
14:00-15:00  数据分析：找爆款规律
15:00-17:00  外包任务跟进 / 新IP研究
19:00-20:00  平台热点研究（热门标签、趋势BGM）
21:00-22:00  发布晚间内容 + 预览
```

### 每日必做清单（Checklist）

#### 内容发布
- [ ] 发布 YouTube Shorts × 2-3条
- [ ] 发布 TikTok × 3-5条
- [ ] 检查所有视频的标题/标签/封面是否优化
- [ ] 在描述中添加联盟链接（如有）

#### 数据监控
- [ ] 记录当日播放量
- [ ] 记录当日收益（平台后台）
- [ ] 记录新增订阅/粉丝数
- [ ] 标记当日爆款视频（播放>1万）
- [ ] 检查版权投诉/警告

#### 互动运营
- [ ] 回复所有评论（24小时内）
- [ ] 点赞/评论同类型账号（增加曝光）
- [ ] 在 Reddit/Pinterest 发布内容引流

#### 优化迭代
- [ ] 分析前日数据：播放量最低的视频，问题在哪里？
- [ ] 分析前日数据：播放量最高的视频，为什么爆了？
- [ ] 调整次日内容策略

## 3. 内容制作 SOP

### 3.1 视频制作标准流程

```
第一步：选题（5分钟）
├─ 查看平台趋势（TikTok Trending / YouTube Shorts）
├─ 确定当日发布的动漫IP
└─ 确定内容类型（解说/剪辑/盘点）

第二步：素材收集（10分钟）
├─ 从原始素材库选取片段
├─ 或使用官方预告片
└─ 记录素材来源（版权合规）

第三步：版权筛查（3分钟）
└─ 运行 copyright_screener.py 检查风险

第四步：字幕翻译（10-30分钟）
├─ 导出中文字幕 SRT
├─ 运行 translate_automation.py 翻译
└─ 校对关键翻译

第五步：剪辑（20-40分钟）
├─ 精剪到 30-90 秒
├─ 叠加字幕
├─ 添加BGM（版权安全音乐库）
└─ 添加水印

第六步：封面制作（5分钟）
└─ 运行 thumbnail_generator.py 或 Canva

第七步：标题 + 标签（5分钟）
├─ 使用 TitleGenerator 生成
├─ 手动微调
└─ 选择15-20个标签

第八步：发布（5分钟/平台）
├─ YouTube Studio / TikTok App
├─ 设置发布时间
└─ 添加描述和联盟链接
```

### 3.2 内容类型优先级

| 类型 | 时长 | 发布频率 | 变现潜力 | 制作难度 |
|-----|------|---------|---------|---------|
| 动漫解说/评析 | 3-10分钟 | 每天1条 | ⭐⭐⭐⭐⭐ | 中 |
| 精彩片段剪辑 | 30-90秒 | 每天5条 | ⭐⭐⭐⭐ | 低 |
| TOP10盘点 | 5-15分钟 | 每周3条 | ⭐⭐⭐⭐ | 中 |
| 壁纸/头像推荐 | 静态 | 每天3条 | ⭐⭐⭐ | 低 |
| 动漫资讯 | 1-2分钟 | 每天2条 | ⭐⭐⭐ | 低 |

### 3.3 爆款内容公式

```
标题公式：
{动漫名} - {EP集数} {爆款描述词} 🔥

爆款描述词：
英文：The Scene That BROKE the Internet / Made Everyone CRY / Left Us SPEECHLESS
印尼：Viral / Bikin Nangis / Membahana
西语：La Mejor Escena / Nos Dejó Locos / Viral Total

封面公式：
红/橙底 + 人物特写(60%) + 大字标题 + 情绪表情(🔥💔)
+ 频道logo水印

开头3秒公式（黄金钩子）：
"这集让所有人都..." → 展示最震撼画面
"最后一幕让粉丝们..." → 悬念感
"终于等到了..." → 期待感
```

## 4. 变现操作 SOP

### 4.1 YouTube Partner Program 申请

**门槛：**
- 1000订阅 + 4000小时观看（长视频）
- 或 1000万Shorts播放（近90天）

**申请步骤：**
```
1. 登录 YouTube Studio
2. 点击左侧菜单"创收"
3. 查看是否达到门槛
4. 点击"开始申请"
5. 阅读并同意 YouTube 创收政策
6. 等待审核（通常1-2个月）
7. 审核通过后设置 AdSense 账号
8. 收款设置（Payoneer/PingPong/Wise）
```

### 4.2 联盟带货设置

**Amazon Associates：**
```
1. 注册 amazonassociates.com
2. 创建联盟链接（每个IP单独创建）
3. 在 YouTube/TikTok 视频描述中插入链接
4. 使用 short link（amzn.to）更美观
5. 每月20日结算（最低$100）
```

**Temu Affiliate：**
```
1. 注册 affiliate.temu.com
2. 获取推广链接和 Banner
3. 在视频描述中添加 Temu 折扣码
4. 设置佣金接收方式
```

### 4.3 版权代理业务起步

**第一步：整理授权清单（Day 1）**
```
列出你想代理的IP：
- 国漫（斗罗大陆、凡人修仙传等）
- 申请渠道（腾讯动漫国际、快看等）
```

**第二步：制作代理方案（Day 2-3）**
```
制作一份简单的"IP海外运营方案"PPT，包含：
- IP介绍（播放量/受众/热度）
- 海外市场分析
- 运营策略
- 收益预估
- 代理佣金方案（20-30%）
```

**第三步：联系版权方（Day 4+）**
```
邮件模板：
Subject: [版权合作] {你的名字} - {IP名} 海外运营代理合作

Dear {版权方},

我是{名字}，专注动漫海外运营代理。

{IP名}在国内已有极高热度，我认为在{目标地区}市场有巨大潜力。

我目前运营着{粉丝数}的动漫账号矩阵，
已成功帮助{其他IP}在海外平台获得{成绩}。

希望与贵方探讨非独家海外代理合作，
我可以负责：内容本地化 + 平台分发 + 运营增长。

期待进一步沟通。

Best,
{名字}
{联系方式}
```

## 5. 授权对接 SOP

### 5.1 版权方联系优先级

```
第一优先级（最快响应）：
├─ 腾讯动漫国际部（国际合作最成熟）
├─ Bilibili International
├─ 快看漫画国际
└─ Webnovel Comics

第二优先级（需商务渠道）：
├─ 动漫CP工作室（微博/B站私信）
├─ 各省动漫协会
└─ 动漫展会商务区（CCG/CICF/ChinaJoy）

第三优先级（长期培养）：
├─ 直接联系作者/画师（Twitter/微博）
└─ 通过MCN机构介绍
```

### 5.2 授权谈判要点

| 要点 | 建议 |
|-----|------|
| 授权范围 | 先谈非独家，避免独家对赌 |
| 期限 | 先签6个月试运行，再谈年度 |
| 地区 | 从单一地区开始（东南亚/中东） |
| 佣金比例 | 20-30%，不要一开始就要独家高价 |
| 内容审批 | 确保有快速审批通道（24小时） |
| 结算周期 | 月结30天或季度结算 |

### 关键数据阈值（触发告警）

- 连续3天播放下跌 → 暂停发布，研究原因
- 收到版权投诉 → 立即下架，申诉
- 收益连续下降 → 调整变现链路
