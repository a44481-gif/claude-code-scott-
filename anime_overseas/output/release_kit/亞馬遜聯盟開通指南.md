## 🚀 5分鐘開通流程

### Step 1：註冊亞馬遜聯盟（3分鐘）

```
1. 打開瀏覽器 → https://affiliate-program.amazon.com
2. 點「Sign Up」
3. 登入你的 Amazon 帳號（沒有的話先註冊一個）
4. 填寫個人信息：
   → 國家/地區：選你所在的國家
   → 網站/移動App：填「YouTube頻道 + TikTok主頁連結」
     （如果沒有網站，就填 YouTube 頻道連結）
   → 主要內容：選「Movies, Anime, TV」
   → 填寫後提交
5. 等待審批（通常幾小時到1天，有時立馬通過）
```

### Step 2：生成凡人修仙傳周邊聯盟連結（2分鐘）

```
1. 登入 Amazon Associates 後台
2. 在頂部搜索框搜：「Fanren Xiuxian Zhuan」或「cultivation anime merch」
3. 找到相關周邊商品（手辦/衣服/海報/書）
4. 點「Get Link」→ 複製短連結（amzn.to 格式）
5. 把這個連結粘貼到你的 YouTube 視頻描述裡
```

**直接連結（可先用這些測試）：**
```
凡人修仙傳周邊搜索：
https://www.amazon.com/s?k=cultivation+anime+merch
https://www.amazon.com/s?k=xuanhuan+anime+figures
```

### Step 3：把連結放進 YouTube 描述（立即執行）

在每條 YouTube 視頻的描述裡，在「Tags」下面添加：

```
💰 Shop Fanren Xiuxian Zhuan Merch:
你的聯盟連結（如 https://amzn.to/xxxxx）
```

## 📊 佣金怎麼算？

| 商品類別 | 佣金比例 | 凡人修仙傳相關商品 |
|---------|---------|----------------|
| 圖書/漫畫 | 4.5% | 凡人修仙傳原作小說 |
| 玩具/遊戲 | 3% | 動漫手辦/人偶 |
| 服裝 | 10% | 凡人修仙傳T恤/外套 |
| 家居/廚房 | 8% | 動漫杯子/抱枕 |

**計算例子：**
- 1件 $25 的凡人修仙傳T恤 → 佣金 $2.50
- 如果每1000播放有2人通過連結購買 → 1000播放 = $5
- 10萬播放 → $500 潛在佣金

## ⚠️ 重要提醒

```
✅ 可以：在描述裡放連結（「購買周邊支持頻道」）
❌ 不可以：在視頻裡直接說「點連結買東西」（這是亞馬遜禁止的）
❌ 不可以：放競爭對手的連結
✅ 正確方式：被動展示「有興趣可以點描述裡的連結」
```

## 🎯 聯盟收益優化技巧

1. **每條視頻只放1個主要連結**（放太多顯得像廣告）
2. **選擇高傭金商品**：服裝/家居 > 圖書/玩具
3. **用短連結**（amzn.to 更美觀，點擊率更高）
4. **在描述前加表情**：💰 符號能提高點擊率
5. **累積粉絲後**：在粉絲>1萬時，可以做「粉絲福利」引導購買
