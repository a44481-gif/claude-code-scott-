## 📱 CapCut 操作流程（15分鐘版）

### 步驟 1：新建項目（2分鐘）
```
1. 打開 CapCut（國際版）
2. 點「+ 新建項目」
3. 點「導入」→ 選「素材庫」
4. 找到下載的凡人修仙傳片段，點選中 → 添加
```

### 步驟 2：裁剪精華（8分鐘）
```
1. 在時間軸上，拖動左右邊框，選取 60-90 秒的精華片段
2. 點「剪裁」→ 放大要留住的畫面部分
3. 原則：戰鬥場面 > 突破場面 > 情感場面
   → 最精彩的一瞬間要在前 5 秒內出現！
```

### 步驟 3：加字幕（3分鐘）
```
1. 點底部「文字」→ 「自動字幕」
2. 等待 CapCut 識別完成
3. 點擊字幕 → 「翻譯」→ 選「英語」
4. 調整字體大小：滑杆拉到 120%
5. 字體顏色：白色，描邊：黑色
6. 位置：畫面下半部中間
```

**凡人修仙傳高情緒台詞翻譯（直接複製粘貼）：**
```
「區區凡人，也敢與我爭鋒？」→ "A mere mortal, daring to challenge me?"
「這不可能！我的實力...」→ "Impossible! My power..."
「我等這一刻，等了一千年」→ "I've waited a thousand years for this moment"
「受死吧！」→ "DIE!"
「讓我來守護你」→ "Let me protect you"
「你永遠也追不上我」→ "You'll never catch up to me"
「今日，我必殺你」→ "Today, I will kill you"
「師傅...我做到了」→ "Master... I did it"
```

### 步驟 4：加 BGM（2分鐘）
```
1. 點「音頻」→ 「音樂」
2. 搜：epic emotional 或 chinese anime
3. 選一個節奏強的，時長要覆蓋整個片段
4. CapCut 內建音樂庫全部免費商用，放心用
```

### 步驟 5：換封面（2分鐘）
```
1. 點底部時間軸最左側的縮略圖
2. 點「設置封面」
3. 從相冊選擇對應的 cover_en_0X.png
4. 或者截圖精彩畫面，手動加文字
```

### 步驟 6：導出（1分鐘）
```
1. 點右上角「導出」
2. 設置：1080p / H.264 / 60fps（或30fps）
3. 保存到相冊
```
