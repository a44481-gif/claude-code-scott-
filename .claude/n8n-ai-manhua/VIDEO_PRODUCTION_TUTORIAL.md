# 方法一：图文视频（最简单，5分钟）

## 工具：Canva + CapCut

---

## 第一步：用Canva制作图片（3分钟）

### 1.1 打开Canva

```
网址：canva.com
注册：用Google账号登录
```

### 1.2 创建项目

```
1. 点击"创建设计"
2. 选择"Instagram Post"（竖屏）
3. 尺寸：1080 x 1920
```

### 1.3 制作封面

**模板结构：**

```
┌──────────────────────┐
│                      │
│    [爆款图片背景]     │
│    可以用渐变/金色背景 │
│                      │
│   ┌────────────┐    │
│   │            │    │
│   │  👑 龙王   │    │
│   │            │    │
│   │ 婚礼现场  │    │
│   │ 被羞辱！   │    │
│   │            │    │
│   │ 全场吓傻！ │    │
│   │            │    │
│   └────────────┘    │
│                      │
│   👇全集已更新      │
│                      │
└──────────────────────┘
```

### 1.4 添加文字

**主标题（最大）：**
```
字体：粗体黑色或金色
大小：80-100px
内容：龙王归来
颜色：#FFD700（金色）或白色
```

**副标题：**
```
字体：常规
大小：40-50px
内容：婚礼现场被羞辱！
颜色：白色
```

### 1.5 保存图片

```
点击右上角"下载"
格式：PNG
分辨率：高质量
保存到桌面
```

---

## 第二步：用CapCut制作视频（2分钟）

### 2.1 下载CapCut

```
手机：应用商店搜索"CapCut"
电脑：capcut.com
```

### 2.2 开始制作

```
1. 打开CapCut
2. 点击"新建项目"
3. 点击"导入"
4. 选择Canva生成的图片
5. 点击"添加到时间轴"
```

### 2.3 添加字幕

```
1. 点击"文字"
2. 点击"添加文本"
3. 输入台词
4. 调整位置和大小
5. 样式：白色粗体，带黑色描边
```

**字幕示例：**
```
第1屏："婚礼现场，所有人都叫我废物..."
第2屏："未婚妻当众退婚！"
第3屏："三千亿将士到场！"
第4屏："他们叫我——龙王！"
```

### 2.4 添加背景音乐

```
1. 点击"音频"
2. 点击"添加音频"
3. 搜索"史诗"或"震撼"
4. 选择合适的音乐
5. 调整音量：30%
```

### 2.5 导出视频

```
1. 点击右上角"导出"
2. 选择分辨率：1080P
3. 选择帧率：30fps
4. 等待导出完成
5. 保存到相册
```

# 方法二：AI生成漫剧图片（推荐）

## 工具：Leonardo AI / Bing Image Creator

---

## 用Bing Image Creator（免费）

### 第一步：生成图片

打开：bing.com/create

**提示词（Prompt）：**
```
中文版：
"6格漫画风格的图片，竖屏排列
第1格：男人被推倒在地，被众人嘲笑
第2格：女人扔戒指，说他不配
第3格：男人低头苦笑
第4格：大门被踢开
第5格：军队进场
第6格：男人戴上墨镜，霸气离开
风格：写实漫画，中国风，震撼"

英文版（效果更好）：
"6 panel manga style comic, vertical layout
Panel 1: Man pushed to ground, crowd laughing at him
Panel 2: Woman throwing ring, saying he's not worthy
Panel 3: Man bowing head, bitter smile
Panel 4: Door kicked open dramatically
Panel 5: Army soldiers marching in
Panel 6: Man putting on sunglasses, walking away powerful
Style: realistic manga, Chinese style, dramatic, 4K"
```

### 第二步：下载图片

```
1. 生成满意后
2. 点击"下载"
3. 保存高清版本
```

### 第三步：用CapCut拼接

```
1. 导入6张图片
2. 每张图片设置2秒时长
3. 添加字幕
4. 添加音乐
5. 导出
```

# 💡 常见问题

## Q：不会画画怎么办？

A：用AI生成图片，不需要画画基础

## Q：没有灵感写台词？

A：用我给你的模板，直接复制

## Q：音乐哪里找？

A：CapCut内置免费音乐，或者抖音热门音乐

## Q：视频上传不了？

A：检查文件大小，抖音限制500MB以内

## Q：定时发布失败？

A：确保手机有电和网络连接
