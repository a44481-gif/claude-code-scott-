# 🎯 方案三：n8n自动化（高级）

## 架构

```
┌─────────────┐
│  定时触发   │  每天自动执行
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  生成内容   │  调用Claude生成
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  制作图片   │  Canva API
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  生成视频   │  CapCut API
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  发布内容   │  TikTok API
└─────────────┘
```

## n8n工作流节点

### 节点1：Schedule Trigger
```javascript
// 每8小时触发一次
{
  "rule": {
    "interval": [
      { "hours": 8 }
    ]
  }
}
```

### 节点2：Code - 生成今日内容
```javascript
// 随机选择题材
const themes = ['龙王', '逆袭', '虐渣', '重生', '神医', '系统'];
const theme = themes[Math.floor(Math.random() * themes.length)];

return {
  theme,
  prompt: `生成${theme}题材漫剧脚本，JSON格式`
};
```

### 节点3：HTTP Request - Claude API
```
URL: https://api.anthropic.com/v1/messages
Method: POST
Headers: { x-api-key: YOUR_KEY }
Body: { messages: [{ content: $json.prompt }] }
```

### 节点4：Discord/Telegram 通知
```
发送生成的内容给你
你确认后手动发布
```
