# 🎯 方法一：图文视频（最简单）

## 步骤1：制作封面（Canva）

```
1. 打开Canva
2. 点击"创建设计"
3. 选择"Instagram Post"
4. 尺寸：1080 x 1920
5. 选择模板或空白设计
```

## 步骤2：设计封面

```
📐 封面结构：
┌──────────────────────┐
│                      │
│   [爆款图片背景]     │
│   金色/红色渐变      │
│                      │
│   ┌──────────────┐  │
│   │              │  │
│   │  👑 龙王    │  │
│   │              │  │
│   │ 婚礼被羞辱  │  │
│   │ 全场吓傻！  │  │
│   │              │  │
│   └──────────────┘  │
│                      │
│   👇全集已更新      │
│                      │
└──────────────────────┘

🎨 颜色：金色(#FFD700) + 深红(#8B0000)
✏️ 字体：粗体白色，带黑色描边
📐 尺寸：1080 x 1920像素
```

## 步骤3：剪辑视频（CapCut）

```
1. 打开CapCut
2. 点击"新建项目"
3. 导入封面图片
4. 点击图片 → 时长设为45秒
5. 添加字幕
6. 添加音乐
7. 导出
```

## 步骤4：添加字幕

```
💬 字幕格式：
- 白色粗体
- 黑色描边
- 居中显示
- 每句2-3秒

📝 示例：
"婚礼现场，所有人都叫我废物..."
"未婚妻当众退婚！"
"三千亿将士到场！"
"——龙王！！！"
```

## 步骤5：添加音乐

```
🎵 来源：CapCut内置
🔍 搜索：史诗/震撼/复仇

📊 音量设置：
- 背景音乐：30%
- 原声：70%
```

# 🎯 方法二：AI生成漫剧图片

## 工具：Bing Image Creator（免费）

### 步骤1：生成图片

打开：bing.com/create

**英文提示词：**
```
"6 panel manga comic, vertical layout
Panel 1: Man humiliated at wedding, pushed to ground
Panel 2: Woman throwing ring, crowd laughing
Panel 3: Man alone, head down, bitter smile
Panel 4: Door kicked open dramatically
Panel 5: Army soldiers marching in
Panel 6: Man wearing sunglasses, walking away powerful
Style: realistic manga, Chinese drama, epic, 4K"
```

### 步骤2：下载图片

```
1. 生成满意后
2. 点击"下载"
3. 保存6张图片
4. 命名：01.jpg, 02.jpg...
```

### 步骤3：拼接视频

```
1. CapCut → 新建项目
2. 导入6张图片
3. 每张图片 → 5秒
4. 添加字幕
5. 添加音乐
6. 导出
```

# 📝 字幕制作规范

## 字幕格式

| 项目 | 规范 |
|------|------|
| 字体 | 黑体/粗体 |
| 大小 | 40-60px |
| 颜色 | 白色 |
| 描边 | 黑色2px |
| 位置 | 居中偏下 |
| 行数 | 每句一行 |

# 📤 导出设置

## 抖音/TikTok标准

| 项目 | 设置 |
|------|------|
| 分辨率 | 1080 x 1920 |
| 帧率 | 30fps |
| 格式 | MP4 |
| 时长 | 30-60秒 |
| 文件大小 | <500MB |

## 导出步骤

```
1. 点击右上角"导出"
2. 选择分辨率：1080P
3. 选择帧率：30fps
4. 等待导出
5. 保存到相册
```

# 📱 抖音发布步骤

## 步骤1：上传视频

```
1. 打开抖音
2. 点击"+"
3. 选择视频
4. 或直接上传已剪辑的视频
```

## 步骤2：设置封面

```
1. 点击"选择封面"
2. 截图或上传封面图
3. 调整封面位置
4. 确认
```

## 步骤3：写标题

```
格式：
[情绪词]场景！下一秒[反转]！[情绪词]！

示例：
婚礼现场被羞辱！下一秒三千亿将士到场！全场吓傻！😭👑
```

## 步骤4：添加标签

```
#龙王 #婚礼反转 #逆袭 #爽剧 #漫剧
#全集解锁 #海外华人 #爆款 #短剧 #热门
```

## 步骤5：定时发布

```
1. 点击"..."更多
2. 选择"定时发布"
3. 设置时间
4. 确认发布
```

## 步骤6：置顶评论

```

👉 链接在主页！
```

# 💡 常见问题

## Q：不会画画怎么办？
A：用AI生成图片，或用Canva模板

## Q：音乐有版权吗？
A：CapCut/剪映内置音乐免费商用

## Q：视频上传失败？
A：检查文件大小（<500MB）

## Q：没有播放量？
A：换封面和标题，继续发

## Q：被搬运怎么办？
A：加自己水印，举报搬运者
