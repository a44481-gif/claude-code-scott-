```
1. 点击"注册"
2. 选择"Google"
3. 选择你的Google账号
4. 允许授权
5. 完成！
```

### 方式3：Apple ID
```
1. 点击"注册"
2. 选择"Apple"
3. 用Apple ID登录
4. 完成！
```
