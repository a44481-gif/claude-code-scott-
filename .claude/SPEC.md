## 1. 业务定位

### 身份与服务
- **角色**: 豆包运营代理人
- **目标客户**: 企业、商家、个人创业者
- **核心服务**:
  - AI 代运营 (月费制)
  - AI 提示词定制
  - AI 脚本开发
  - AI 数字人视频制作
  - 自动化工作流搭建

### 商业模式
- 轻资产运营
- 服务 + 模板 + 工具包
- 可规模化、可重复销售
- 订阅制 + 项目制双轨变现

## 3. 支付宝收款实现

### 收款方式
1. **个人收款**: 支付宝当面付（个人可申请）
2. **商户收款**: 支付宝商户 API（企业/个体户）
3. **转账码**: 生成固定金额收款码

### 技术实现
- 订单创建 → 支付链接/二维码生成 → 用户支付 → 回调通知 → 订单校验
- 支持自定义金额和固定金额
- 支付成功自动触发交付流程

## 4. 获客与运营自动化

### 平台引流
- 抖音/小红书/视频号: 短视频+图文种草
- 知乎: 干货文章引流
- 闲鱼: 服务型商品上架

### 自动化流程
- 需求收集 → 自动回复 → 报价发送 → 支付引导 → 服务交付

## 5. 交付与售后

### 自动交付
- 付款成功 → 发送服务链接/文件 → 开通权限
- 自动发送合同、教程、售后说明

### 售后支持
- FAQ 自动回复
- 工单系统
- 定期回访

## 7. 技术架构

### 前端 (Web落地页)
- React + TypeScript
- TailwindCSS
- 移动端优先

### 后端 (Python)
- Flask/FastAPI
- 支付宝 SDK
- 数据库: SQLite/MySQL
