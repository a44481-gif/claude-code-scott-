### 2.2 D-ID API 调用示例

```python
import requests

def create_zen_sister_video(script, image_url, audio_url):
    """创建禅心师姐说话视频"""
    api_url = "https://api.d-id.com/talks"

    payload = {
        "source_url": image_url,
        "script": {
            "type": "text",
            "input": script,
            "provider": {
                "type": "microsoft",
                "voice_id": "zh-CN-XiaoxiaoNeural"  # 或使用克隆声音
            }
        },
        "config": {
            "fluent": True,
            "pad_audio": "0.5"
        }
    }

    headers = {
        "Authorization": "Bearer YOUR_D-ID_API_KEY",
        "Content-Type": "application/json"
    }

    response = requests.post(api_url, json=payload, headers=headers)
    return response.json()
```

## 三、声音克隆提示词

### 3.1 ElevenLabs 声音克隆配置

```
【声音样本要求】
- 样本数量：3-5个音频文件
- 总时长：至少10-15分钟
- 内容：包含各种情感和语速的语音
- 格式：MP3 / WAV，48kHz
- 质量：安静环境，无背景音乐，吐字清晰

【声音描述（用于克隆）】
Voice Description:
- Gender: Female
- Age: Late 20s to early 30s
- Tone: Warm, gentle, slightly low-pitched
- Quality: Soft and airy with a sense of calm
- Speaking style: Slow and deliberate, with intentional pauses
- Accent: Standard Mandarin Chinese
- Emotional range: Calm, reassuring, wise but not preachy
- Unique characteristics: End of sentences sometimes trails off
  softly, like a gentle exhale

【推荐微调参数】
- Stability: 0.7 (较高稳定性，保持一致)
- Similarity: 0.8 (较高相似度)
- Style: 0.3 (低风格化，自然为主)
- Speaker Boost: On
```

### 3.2 讯飞声音克隆配置

```
【录音规范】
- 环境：安静室内，回声小于40dB
- 设备：手机录音即可，AirPods或有线麦克风
- 采样：44.1kHz / 16bit
- 语速：中速偏慢，停顿清晰
- 内容：包含声母韵母全覆盖，10-50字/句

【音色特征描述】
- 音色：温柔低沉女声
- 性别：女
- 年龄感：25-35岁
- 语言：普通话，一乙以上
- 特点：说话有禅意，停顿间有气声
- 情感：平静、温暖、治愈

【合成参数】
- 基础音色：温柔女声
- 语速：稍慢（0.85-0.95）
- 音调：标准偏低
- 情感类型：温暖治愈
```

## 五、数字人保养与更新

### 5.1 形象一致性维护
```
- 保持同一套形象/服装/场景
- 记录每次生成的Prompt，便于复用
- 定期更新背景/造型，保持新鲜感
```

### 5.2 声音一致性维护
```
- 保持同一克隆声音
- 不同内容类型的语速/语调参数存档
- 避免混用其他声音
```

### 5.3 定期优化
```
每月：
- 收集反馈，分析粉丝反应
- 调整形象/声音参数
- 测试新的场景/造型
- 优化制作效率
```

## 六、常见问题解决

| 问题 | 解决方案 |
|------|---------|
| 面部不自然 | 使用更高质量的形象图，确保正面光照 |
| 声音不同步 | 调整唇形同步参数，或重新生成配音 |
| 背景穿帮 | 使用纯色背景，或后期抠像更换 |
| 动作僵硬 | 选择更流畅的驱动模式 |
| 生成失败 | 检查API额度，尝试更换时段 |
