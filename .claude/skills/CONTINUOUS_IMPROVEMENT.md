# 持續改進機制

## 每日自動化流程

### 早上 9:00 (已設置)
- 自動運行技能統計報告
- 檢查待處理反饋
- 識別改進點

---

## 手動優化流程

### 每次任務後
```
1. 記錄使用情況
   python skill_optimizer.py use success <skill_name>

2. 收集反饋（如有問題）
   python skill_optimizer.py feedback <skill> <問題描述>

3. 記錄學習
   python skill_optimizer.py learn <教訓內容>
```
