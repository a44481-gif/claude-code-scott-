```
SMTP_PORT=465

```

### 獲取163授權碼

1. 登入 [mail.163.com](https://mail.163.com)
2. → 設置 → POP3/SMTP/IMAP
3. → 開啟 IMAP/SMTP 服務
4. → 點擊"開啟"後會顯示**授權碼**（16位）
5. 將授權碼填入 `SMTP_PASSWORD`
