## ⏰ 定时任务

已设置Windows任务计划：
- 任务名：漫剧出海每日提醒
- 执行时间：每天 08:00
- 执行内容：弹出提醒窗口

查看任务：
```batch
schtasks /query /tn "漫剧出海每日提醒"
```

删除任务：
```batch
schtasks /delete /tn "漫剧出海每日提醒" /f
```
