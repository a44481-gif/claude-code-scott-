# 📋 自动化方案

## 方案1：Browser Automation（推荐）

使用 Playwright 自动化浏览器操作

### 安装步骤

```batch
# 1. 安装 Node.js (如果没有)
# https://nodejs.org/

# 2. 安装 Playwright
cd manhua-system
npm init -y
npm install playwright

# 3. 安装浏览器
npx playwright install chromium
```

### 运行自动化

```batch
cd manhua-system
node scripts/auto-poster.js
```
