# 抖音直播推流地址获取图文教程

## 方法一：直接访问直播中控台

### 步骤 1：打开直播中控台
直接访问这个链接：
```
https://live.douyin.com/
```

登录你的抖音账号即可。

---

## 方法二：从创作者服务平台获取

### 步骤 1：打开创作者服务平台
```
https://creator.douyin.com/
```

### 步骤 2：登录账号
- 点击右上角「登录」
- 用抖音APP扫码登录

### 步骤 3：找直播功能
登录后，在左侧菜单找：

```
创作者中心
├── 直播管理  ← 点击这里
├── 作品管理
└── 数据中心
```

**如果找不到直播管理：**

1. 点击「全部功能」
2. 搜索「直播」
3. 点击「直播中控台」

---

## 方法三：手机端获取

### 步骤 1：打开抖音APP
### 步骤 2：点击右下角「我」
### 步骤 3：点击右上角「≡」
### 步骤 4：找「创作者服务」→「直播」
### 步骤 5：点击「电脑直播」

---

## 获取推流地址

进入电脑直播页面后：

```
┌─────────────────────────────────────┐
│                                     │
│      【电脑直播】                   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  推流地址                    │   │
│  │  rtmp://push.toutiao.com/.. │   │
│  │  [复制]                      │   │
│  └─────────────────────────────┘   │
│                                     │
│  复制全部地址给我即可               │
│                                     │
└─────────────────────────────────────┘
```

点击「复制」按钮，复制完整的推流地址。

## 常见问题

### Q: 找不到电脑直播选项？
A: 可能需要先完成实名认证。在「设置」→「账号与安全」→「实名认证」中完成认证。

### Q: 推流地址灰色不可复制？
A: 需要先开启直播功能，可能需要粉丝数达到一定数量。

### Q: 没有直播管理菜单？
A: 尝试刷新页面，或清除缓存后重新登录。
