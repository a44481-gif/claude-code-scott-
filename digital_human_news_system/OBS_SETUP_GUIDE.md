# OBS直播配置指南

## 1. 下載與安裝

### 下載OBS Studio
- 官網: https://obsproject.com/
- 下載完成後安裝

### 安裝虛擬攝像頭插件 (可選)
- OBS-VirtualCam
- 用於將OBS輸出作為攝像頭使用

---

## 2. 配置數字人場景

### 創建場景
1. 點擊「場景」區域的「+」
2. 輸入場景名稱：`數字人直播`
3. 點擊「確定」

### 添加來源
1.點擊「來源」區域的「+」

2. **數字人視頻源** (選擇其一):

   **方案A: 本地視頻循環**
   - 選擇「媒體源」
   - 選擇AI數字人生成的視頻文件
   - 勾選「循環」

   **方案B: 虛擬攝像頭**
   - 選擇「虛擬攝像頭」
   - 選擇數字人軟件

   **方案C: 圖片+文字**
   - 添加「圖像」作為背景
   - 添加「文字 (GDI+)」作為字幕

3. **添加字幕文字源**
   - 選擇「文字 (GDI+)」
   - 名稱: `字幕`
   - 調整字體大小位置

4. **添加標題文字源**
   - 選擇「文字 (GDI+)」
   - 名稱: `標題`

---

## 3. 配置音頻

1. 在「音頻混音器」中:
   - 麥克風: 選擇你的麥克風
   - 桌面音頻: 選擇系統音頻

2. 如需AI語音:
   - 添加「應用音頻捕獲」
   - 或使用數字人軟件的音頻輸出

---

## 4. 配置推流

### 抖音直播
1. 登錄抖音創作服務平台
2. 進入「直播間管理」
3. 複製推流地址

### OBS設置
1. 點擊「設置」
2. 選擇「推流」
3. 服務器: `自定義`
4. 服務器填入抖音推流地址 (去掉結尾的stream key)
5. 推流密鑰填入抖音給的stream key

### B站直播
1. 登錄B站直播間
2. 获取推流地址
3. OBS推流設置:
   - 服務器: `rtmp://live-push.bilivideo.com/live-bvc`
   - 推流密鑰: 填入B站提供的密鑰

## 6. 快捷鍵設置

| 功能 | 快捷鍵 |
|------|--------|
| 開始推流 | Ctrl+Shift+S |
| 停止推流 | Ctrl+Shift+D |
| 靜音 | Ctrl+M |
| 隱藏字幕 | Ctrl+1 |

## 7. 自動化腳本

### 自動開始推流腳本
```bash
# 將以下內容保存為 start_stream.bat
start "" "obs64.exe" --startstreaming
```

### 自動停止推流腳本
```bash
# 將以下內容保存為 stop_stream.bat
taskkill /f /im obs64.exe
```

---

## 8. 排查問題

### 推流失敗
1. 檢查推流地址是否正確
2. 確認網絡穩定
3. 嘗試降低碼率

### 卡頓
1. 降低解析度至1280x720
2. 降低碼率至2500-4000
3. 關閉其他佔用帶寬的程序

### 數字人卡頓
1. 預先渲染完整視頻
2. 使用硬件加速
3. 關閉不必要的效果

---

## 9. 推薦設定

### 解析度
- 1080p: 1920x1080 (推薦)
- 720p: 1280x720 (低配電腦)

### 碼率
- 1080p: 4000-6000 kbps
- 720p: 2500-4000 kbps

### 幀率
- 30fps (標準)
- 60fps (高質量)

### 音頻
- 比特率: 192 kbps
- 採樣率: 48kHz
