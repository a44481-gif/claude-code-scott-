# 云端协同架构方案

## 架构概览

```
┌─────────────────────────────────────────────────────────────┐
│                      云端 (Cloud)                          │
│  ┌─────────────────────────────────────────────────────┐  │
│  │              Claude Code / n8n Cloud                │  │
│  │  ┌──────────┐  ┌──────────┐  ┌──────────┐         │  │
│  │  │ 新闻抓取 │→ │ AI二创   │→ │ 视频生成  │         │  │
│  │  └──────────┘  └──────────┘  └──────────┘         │  │
│  │                      ↓                            │  │
│  │               ┌──────────┐                       │  │
│  │               │ 结果存储 │                       │  │
│  │               │ (JSON)   │                       │  │
│  │               └──────────┘                       │  │
│  └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                           ↓ 推送/Webhook
┌─────────────────────────────────────────────────────────────┐
│                      本地 (Local)                           │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                │
│  │ 结果接收 │→ │ 数据存储 │→ │ 邮件通知 │                │
│  │ Webhook  │  │ SQLite   │  │ 用户判定  │                │
│  └──────────┘  └──────────┘  └──────────┘                │
└─────────────────────────────────────────────────────────────┘
```

## 云端服务选项

| 方案 | 服务 | 特点 |
|------|------|------|
| **方案A** | n8n Cloud | 免费、易用、无需服务器 |
| **方案B** | 飞书云空间 | 团队协作、数据存储 |
| **方案C** | 腾讯云函数 | 自动扩缩容、按需付费 |
| **方案D** | 本地Webhook + 外部访问 | 简单直接 |

## 推荐方案：n8n Cloud + 本地Webhook

### 优势
- n8n Cloud 免费额度足够
- 无需管理服务器
- 定时执行 + Webhook 触发
- 结果自动推送

### 实施步骤

1. 注册 n8n Cloud (https://n8n.io/cloud)
2. 导入工作流
3. 配置 Webhook 推送回本地
4. 本地接收服务存储

---

## 数据流向

```
云端执行 → 生成结果 → Webhook推送 → 本地接收 → SQLite存储 → 邮件通知
```

## 安全考虑

- Webhook 签名验证
- 数据加密传输
- 本地防火墙配置
