# 🎬 爪爪博士 - 7天视频脚本

> 每天发布1条，持续7天，快速起号！
> 时长：15-45秒 | 风格：萌趣专业 | 发布平台：抖音/TikTok

## 📅 Day 1 - 开场介绍

### 《爪爪博士来了》

```
【开场钩子】0-3秒
画面：可爱猫咪/狗狗眨眼
声音：喵~ 你知道你的宠物在想什么吗？

【自我介绍】3-15秒
画面：爪爪博士出镜（可爱形象）
声音：大家好，我是爪爪博士！
      一个懂宠物心理的博主。
      养宠物这么久，
      你真的了解它们吗？

【价值预告】15-25秒
画面：萌宠可爱瞬间合集
声音：接下来的日子，
      我会分享宠物心理学、
      训练技巧、健康知识，
      让你的毛孩子更幸福！

【互动引导】25-30秒
画面：博士和宠物互动
声音：你是猫奴还是狗奴？
      评论区告诉我！
      关注我，一起科学养宠！
```

**标签**：#爪爪博士 #养宠知识 #宠物心理 #猫奴 #狗奴

## 📅 Day 2 - 知识科普

### 《猫咪爱你的5个表现》

```
【开场钩子】0-3秒
画面：猫咪舔爪子特写
声音：你的猫爱你吗？
      答案在这里！

【内容】3-35秒
画面：可爱猫咪 + 文字说明
声音：
第一，猫咪蹭你。
  代表它在说：你是我的！

第二，猫咪对你露肚子。
  恭喜，这是最大的信任！

第三，猫咪慢慢眨眼。
  这等于在说：我爱你。

第四，猫咪带"礼物"给你。
  老鼠、虫子...
  虽然吓人，但这是爱！

第五，猫咪踩奶。
  这是小时候对猫妈妈的记忆，
  说明它把你当妈妈了！

【结尾】35-40秒
画面：猫咪蹭镜头
声音：中了几个？
      评论区告诉我！
```

**标签**：#猫奴 #猫咪爱你 #喵星人 #宠物行为 #养猫知识

## 📅 Day 3 - 趣味测试

### 《假装睡着测试狗狗反应》

```
【开场钩子】0-3秒
画面：狗狗在主人旁边
声音：假装睡着，看看狗狗会怎么做？

【测试过程】3-25秒
画面：第一视角拍摄
声音：
（假装睡觉，闭眼不动）

5秒后...
狗狗开始东张西望。

10秒后...
它走过来，闻了闻你。

15秒后...
（如果狗狗舔脸/叫/蹭）
哇！它超担心你的！

【结果揭晓】25-30秒
画面：主人睁眼，狗狗反应
声音：狗狗的反应，
      说明它有多爱你！

【引导】30-35秒
画面：狗狗开心的脸
声音：你家狗狗是怎么反应的？
      评论区分享！

（如果是猫...猫可能看都不看你）
```

**标签**：#狗狗 #宠物测试 #狗奴 #萌宠 #可爱

## 📅 Day 4 - 实用干货

### 《新手养猫最易犯的5个错》

```
【开场钩子】0-3秒
画面：可爱小奶猫
声音：新手养猫？千万别犯这5个错！

【内容】3-35秒
画面：图文配合
声音：
第一，买太小的奶猫。
  不到2个月离开猫妈妈，
  免疫力差，容易生病。

第二，随意换粮。
  换粮要循序渐进，
  7天过渡期，懂了？

第三，不打疫苗。
  猫三联、狂犬疫苗，
  每年都要补！

第四，不封窗。
  猫会跳楼！
  真的会发生！

第五，遛猫。
  猫不需要遛！
  出门容易应激！

【结尾】35-40秒
画面：健康的猫咪
声音：以上5个错，你中了几个？
      评论区聊聊！
```

**标签**：#新手养猫 #养猫知识 #科学养宠 #猫咪健康 #铲屎官

      可爱就够了！

【结尾】40-45秒

## 📅 Day 7 - 互动引导

### 《你家宠物叫什么名字？》

```
【开场钩子】0-3秒
画面：各种宠物的名字牌
声音：你家宠物叫什么名字？
      它的名字有什么故事？

【引导互动】3-30秒
画面：博士拿着问题卡片
声音：
名字是随便取的？
还是有特别的意义？

比如：
- 叫"旺财"因为希望它带来财运？
- 叫"团子"因为它圆滚滚？
- 还是看了什么电影临时起的？

【号召】30-40秒
画面：博士和宠物一起看镜头
声音：评论区告诉我！
      点赞过1000我就去探访！
      关注我，看看有没有你认识的名字！

【结束】40-45秒
画面：宠物挥手
声音：下期见！
```

**标签**：#宠物名字 #互动 #评论区 #铲屎官
