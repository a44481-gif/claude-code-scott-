# 🎬 禅心师姐 - 7天视频脚本

> 每天发布1条，持续7天，快速起号！
> 时长：30-60秒 | 风格：禅意治愈 | 发布平台：抖音/TikTok

---

## 📅 Day 1 - 开场介绍

### 《禅心师姐来了》

```
【开场钩子】0-3秒
画面：禅意书房，日光洒落，师姐侧脸特写
声音：心静了，一切就对了。

【自我介绍】3-15秒
画面：师姐转身，正面微笑
声音：大家好，我是禅心师姐。
      一个用禅意生活的人。
      每天清晨，我想和你一起，
      用一段禅语，开启美好的一天。

【价值预告】15-25秒
画面：师姐沏茶/整理书架
声音：接下来的日子里，
      我会分享禅学智慧、修行心得，
      和让内心平静的方法。

【互动引导】25-30秒
画面：师姐双手合十
声音：如果你也想找到内心的宁静，
      关注我，我们一起修行。

【结束语】30秒
画面：师姐微笑点头
声音：阿弥陀佛。
```

**标签**：#禅心师姐 #禅语 #心灵疗愈 #修行 #正念

## 📅 Day 2 - 禅语智慧

### 《师父教我的三句话》

```
【开场钩子】0-3秒
画面：特写师父背影/寺庙/佛像
声音：出家20年的师父，教会我三句话。

【第一句】3-15秒
画面：师姐站在窗边，望着远方
声音：第一句，
      放下，不是放弃。
      放下心中的执念，
      不是放弃努力，
      而是学会释怀。

【第二句】15-27秒
画面：师姐在花园浇花
声音：第二句，
      一切都是最好的安排。
      无论发生什么，
      都是该发生的。

【第三句】27-40秒
画面：师姐在禅垫上静坐
声音：第三句，
      活在当下。
      过去不可得，未来不可知，
      只有此时此刻，才是真实的。

【总结】40-50秒
画面：日出/山景/平静水面
声音：记住这三句话，
      你的心，会越来越平静。
```

**标签**：#师父 #修行 #禅语 #放下 #活在当下

## 📅 Day 3 - 知识科普

### 《道家的三个智慧》

```
【开场钩子】0-3秒
画面：道观/老子像/太极图案
声音：道家说，做到这三点，好运自然来。

【第一点】3-18秒
画面：流水/云雾/自然风景
声音：第一，顺其自然。
      不是躺平，
      而是尽力而为，顺势而为。

【第二点】18-33秒
画面：师姐在竹林中漫步
声音：第二，返璞归真。
      越简单，越快乐。
      减少欲望，增加幸福。

【第三点】33-48秒
画面：夕阳/宁静的湖面
声音：第三，上善若水。
      最高的善，像水一样，
      利万物而不争。

【结尾】48-55秒
画面：师姐微笑
声音：这就是道家智慧。
      简单，但很深刻。
```

**标签**：#道家 #老子 #道家智慧 #传统文化 #返璞归真

## 📅 Day 4 - 情感共鸣

### 《写给很累的你》

```
【开场钩子】0-5秒
画面：城市霓虹/匆忙的人群
声音：嘿，你累了吗？

【共情】5-15秒
画面：师姐安静坐着，目光温柔
声音：我知道，你很努力。
      每天早出晚归，
      扛着责任，忍着委屈。

【理解】15-28秒
画面：师姐望向窗外
声音：生活不易，我们都懂。
      但我想告诉你，
      你已经很棒了。

【安慰】28-40秒
画面：师姐沏茶，茶香袅袅
声音：累了，就休息一下。
      渴了，就喝杯热茶。
      允许自己脆弱，
      这不是软弱，这是真实。

【鼓励】40-52秒
画面：日出/温暖阳光
声音：记住，
      无论发生什么，
      太阳每天都会升起。
      而你，比你想象的更强大。

【祝福】52-58秒
画面：师姐双手合十
声音：愿你今天，心安，自在。
```

**标签**：#治愈 #累了就休息 #正能量 #情感共鸣 #自我疗愈

## 📅 Day 5 - 实用干货

### 《睡前冥想3分钟》

```
【开场钩子】0-3秒
画面：月亮/星空/安静的夜晚
声音：睡不着？试试这个方法。

【引导】3-40秒
画面：师姐躺在舒适位置，闭眼
声音：现在，闭上眼睛。
      深呼吸，吸气...
      呼气...
      把所有的烦恼，
      都呼出去。

      想象自己，
      躺在一片温暖的草地上，
      阳光洒在身上，
      微风轻轻吹过。

      此刻，没有压力，
      没有焦虑，
      只有平静和安宁。

      你做得很好。
      今晚，好好休息。
      明天，又是美好的一天。

【结束】40-45秒
画面：清晨阳光
声音：晚安。
```

**标签**：#冥想 #失眠 #睡前冥想 #放松 #助眠

## 📅 Day 6 - 故事分享

### 《峨眉山奇遇》

```
【开场钩子】0-5秒
画面：峨眉山云海/寺庙/佛像
声音：五年前，我在峨眉山，
      遇到一位百岁僧人...

【故事】5-40秒
画面：师姐回忆的视角（虚化背景）
声音：那天，我一个人爬山，
      累了，就在一棵大树下休息。

      突然，一位老僧人走过来，
      递给我一杯茶。

      他说：施主，我等你很久了。

      我愣住了。

      他笑着说：
      每个人都会来这里，
      只是时间早晚的问题。

      他给我泡了一杯茶，
      临走时说了一句话：
      「心有禅意，处处是道场。」

【感悟】40-50秒
画面：师姐微笑
声音：从那以后，
      我不再执着于形式。
      修行，不在庙里，
      在每一个当下。

【结尾】50-55秒
画面：阳光穿透树叶
声音：愿你找到自己的道场。
```

**标签**：#峨眉山 #僧人 #故事分享 #禅意 #修行
