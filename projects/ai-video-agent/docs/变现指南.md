## 💚 第一步：绑定收款方式

### 抖音收款绑定

```
1. 打开抖音APP
2. 点击右下角「我」
3. 点击右上角「☰」
4. 点击「设置」
5. 点击「账号与安全」
6. 点击「第三方账号绑定」或「支付方式」
7. 绑定你的支付宝/微信/银行卡
```

### TikTok 收款绑定

```
1. 打开 TikTok APP
2. 点击「Profile」
3. 点击「☰」或「Settings」
4. 点击「Balance and Payments」或「Creator Tools」
5. 设置收款方式：
   - PayPal（推荐）
   - 银行转账（需要美国银行账户）
```

### YouTube 收款绑定

```
1. 登录 YouTube Studio
2. 点击「创收」
3. 点击「设置」
4. 在 AdSense 中设置收款地址
5. 每月21-26日结算
```

## 📱 第二步：开通变现功能

### 抖音创作者激励计划

**开通条件：**
- 粉丝 ≥ 1000
- 实名认证
- 账号状态正常

**开通步骤：**
```
抖音APP → 我 → ☰ → 创作者中心
→ 变现工具 → 创作者激励计划 → 申请开通
```

**收益计算：**
| 内容类型 | 每万播放收益 |
|---------|-------------|
| 宠物类 | ¥30-80 |
| 正能量/禅意 | ¥20-50 |
| 故事类 | ¥40-100 |

### 抖音商品橱窗

**开通条件：**
- 粉丝 ≥ 1000
- 实名认证
- 发布 ≥ 10条视频

**开通步骤：**
```
抖音APP → 我 → ☰ → 创作者中心
→ 变现工具 → 商品橱窗 → 申请开通
```

**带货佣金：**
| 商品类型 | 佣金比例 |
|---------|---------|
| 宠物食品 | 15-30% |
| 宠物用品 | 20-40% |
| 冥想/禅修周边 | 25-50% |

### TikTok Creator Fund

**开通条件：**
- 粉丝 ≥ 10000
- 近30天播放 ≥ 50000
- 18岁以上
- 美国/英国/欧盟用户

**收益：** 每百万播放约 ¥50-150
