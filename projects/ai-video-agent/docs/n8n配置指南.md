
## ⏰ 修改执行时间

如需修改执行时间：

1. 点击 **"Schedule Trigger"** 节点
2. 修改 Cron 表达式：
   - `0 9 * * *` = 每天9点
   - `0 12 * * *` = 每天12点
   - `0 */6 * * *` = 每6小时

## 🔧 高级配置

### 添加 Webhook 触发

如需手动触发，可添加 Webhook 节点：

1. 添加 **"Webhook"** 节点
2. 设置路径：`/daily-content`
3. 复制 Webhook URL
4. 可在任何地方访问该 URL 触发工作流

## 📱 监控运行状态

查看工作流运行历史：
1. 点击工作流名称
2. 选择 **"Executions"** 标签
3. 查看每次运行的详情

## ❓ 常见问题

### Q: n8n 启动失败？
```powershell
# 清理缓存后重试
npm cache clean --force
n8n
```

### Q: 邮件发送失败？
检查 SMTP 配置是否正确，确保授权码有效

### Q: 想停止工作流？
点击右上角 **"Deactivate"** 按钮
